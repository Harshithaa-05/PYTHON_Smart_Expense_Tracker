#  Expense Tracker

##  Project Description

This project is a Python-based command-line application designed to manage and analyze daily expenses. It helps users track their spending habits, identify high-expense categories, and visualize data using charts.

##  Tools Used

* Python
* CSV (for data storage)
* Matplotlib
* OS Module
* Collections (defaultdict)

##  Files

* main.py
* expenses.csv

##  Features

* Add and store daily expenses
* Generate monthly expense summary
* Analyze category-wise spending
* Provide suggestions for overspending
* Visualize expenses using pie charts

##  Key Insights

* Monthly spending trends
* Highest spending category
* Category-wise expense distribution
* Identification of overspending areas

## How to Run

1. Make sure Python is installed
2. Install required library:
   pip install matplotlib
3. Run the program:
   python main.py
